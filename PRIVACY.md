## Data Collection

Roma1990 Pomodoro Timer **does not collect, use, or share any user data**. No personal information, browsing history, or usage analytics are gathered.

## Local Storage

The extension stores the following data **locally on your device only** using Chrome's `chrome.storage.local` API:

- Timer state (duration, start time, running or idle status)
- Daily completed session count

This data is used solely to operate the timer. It **never leaves your browser** and is not accessible to the developer, any third party, or any external server.

## Data Sharing

No user data is shared with any party. There are no third-party services, analytics providers, advertising networks, or external servers involved.

## Permissions Used

| Permission | Purpose |
|---|---|
| `alarms` | Triggers a one-time alarm when the Pomodoro timer completes |
| `storage` | Saves timer state and session count locally on your device |
| `notifications` | Displays a browser notification when a session completes |

No permission is used to access, collect, or transmit user data.

Any updates to this privacy policy will be reflected here with a revised date.
